游戏名称: 飞机大战
英文名称: Airplane War
游戏类型: 弹幕射击类游戏
游戏版本: 1.00a


(1)游戏操作介绍
    ①注册、登录用户后，点击"开始游戏"按钮，触碰屏幕即可开启游戏。

    ②进入游戏后，我方飞机自动跟随鼠标移动。通过移动鼠标进行攻击敌人、躲避敌人射击的操作。

    ③击败敌人"补给飞机"，100%掉落补给品。（血包权重:1，能量权重1，加分星星权重2）
      击败敌人"大飞机"，33.3%掉落补给品。（血包权重:1，能量权重1，加分星星权重2）
      击败敌人"小boss飞机"，100%掉落补能量补给。

    ④己方飞机初始能量为0。拾取能量补给时，能量条+1。
      能量为0时，飞机不间断向前方射击子弹，此时该弹道频率为100%。
      能量为1时，飞行交替向左右1弹道射击子弹，一左一右，此时每边1弹道频的率为50%。
      能量为2时，飞行同时向左右1弹道射击子弹，此时每边1弹道频的率为100%。
      能量为3时，飞行交替向左右2弹道射击子弹，一左一右，此时每边2弹道频的率为50%。
      能量为4时，飞行同时向左右2弹道射击子弹，此时每边2弹道频的率为100%。
      (注意：高能量会完全继承低能量时的弹道，并添加新的弹道子弹；我方外观也会随着能量变换)

    ⑤能量>=4时（飞机变为超级状态、即将出现浅蓝色能量条），通过点击"z"或"Z"，可释放技能，之后能量扣除4点。
      Z键技能为大范围穿透的凤凰子弹，穿透总伤25点血，击败敌人对应血量后凤凰消失。
      技能期间总共可以发射12发凤凰子弹，之后技能自动结束。

    ⑥游戏进行期间，可随时点击"空格"暂停游戏。

    ⑦当我方死亡，或敌方最终boss被击败，游戏结束，分数会自动被计入排行榜。
      点击屏幕结束游戏，自动跳转到结束界面，可查看玩家分数排行。


(2)游戏界面介绍
    ①登录、注册界面
        当程序运行，首先打开登录注册界面，可以选择注册用户，也可以直接登录。
        注册页面所写的注册信息，会被自动传入本机数据库的注册信息表中。
        登录后，进入开始菜单，可以选择开始游戏，也可以先查看该用户的排行。
        用户界面的背景颜色跟随游戏的背景自动变换。

    ②游戏界面
         游戏的背景是四种环绕随机一个（山林/天空/火山/沙漠）
         游戏背景会随着飞机的飞行自动向后移动，并完成背景循环。
         游戏开始后，会循环播放bgm，游戏结束时截至。
         游戏界面中，左上角会打印我方飞机的飞行距离和分数。
         进入游戏后，会绘制我方飞行的血条和能量条。
         大小boss出场后，会绘制对应颜色的能量条。(小boss:红; 大boss:黄)

    ③游戏结束界面
        游戏结束时，会自动将分数录入本机数据库的游戏分数排行表中。
        游戏结束后，单击屏幕进入游戏结束界面，可选择查看分数。
        该界面的背景颜色跟随游戏的背景自动变换。


(3)游戏单位介绍
    ①我方飞机
        初始血量:30:上限:无限
        初始能量:0:上限:无限
        普通子弹级别:0~4级，子弹数量会随着能量增加，具体介绍见"游戏操作介绍"
            0级子弹:垂直向上
            1、2级子弹:偏转角度--26.5度
            3、4级子弹:偏转角度--45度
        超级子弹:穿透总伤--25，是凤凰飞行的动图

    ②敌方小飞机
        初始血量:1
        入场方式:从屏幕上方随机位置入场
        子弹:不发射子弹
        击败后分数+1
        注意:敌方小飞机自身具有跟踪功能

    ③敌方大飞机
        初始血量:3
        入场方式:从屏幕上方随机位置入场
        子弹:发射直线向前飞行的子弹
        分数:击败后分数+3

    ④敌方补给飞机
        初始血量:1
        入场方式:从屏幕上方随机位置入场
        包含3中补给:血包、能量包、分数星星
        血包效果:使我方飞机血量+3
        能量包效果:使我方飞机能量+1
        分数星星效果:使我方分数+7

    ④敌方小boss
        初始血量:200
        入场方式:从屏幕上方正中心入场，之后左右移动
        子弹:三连发，发射发射直线向前飞行的子弹
        击败后必定掉落能量包
        分数:击败后我方分数+15

    ⑤敌方大boss
        初始血量:500
        入场方式:从屏幕上方正中心入场，之后左右移动
        子弹1:二连发，左右各发射直线向前飞行的子弹
        子弹2:单发，会跟踪的子弹
        分数:击败后我方分数+35
        击败后，游戏胜利
